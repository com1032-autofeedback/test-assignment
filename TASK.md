Exercise 2.1
Write a Java program that prints out the size (number of bytes) of primitive date types: short,
char, int, long, float and double.
All primitive values have a predefined size, e.g., short is 2 bytes, char is 2 bytes, int is 4
bytes, float is 4 bytes, double is 8 bytes, long is 8 bytes.
[Hint: you can use the SIZE constant from wrapper class, e.g., Short.SIZE, Long.SIZE]
